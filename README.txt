Beschreibung:
Dieses Programm liest Zahlen aus einer Datei durations.txt aus. Diese Zahlen werden als Sekunden verwendet, die das Programm wartet. 
Am Ende jeder Wartezeit produziert das Programm ein Geräusch. Dies soll dabei unterstützen, beim Augentraining zu wissen, wann man 
die Augenstellung wechseln kann. 

Benutzung:
Sie müssen im selben Ordner, in welchem sich die Datei Augentraining.exe befindet, die Datei durations.txt erstellen. Falls sie 
Augentraining.exe starten ohne diese Datei, wird es sie selbstständig anlegen. durations.txt muss dann entsprechend mit den Zeiten
in Sekunden gefüllt werden. Achten sie dabei darauf, die Zahlen in einer beliebigen Weise voneinander zu trennen. 
Wenn sie ihre Zeiten in durations.txt eingetragen haben, wird das Programm im Konsolenfenster laufen und ihnen die aktuelle
Zeit anzeigen. 

Code:
Das Programm wurde in C++ geschrieben. Der Code liegt bei, falls sich jemand das Programm maßschneidern möchte.
